# 🔥 Automatic and Manual Controlled Firefighting Robot

This project is an IoT-enabled firefighting robot built using an **ESP32 DevKit V1**, **L298N motor driver**, and **flame sensors**.  
It can operate in both **automatic** and **manual (esp web server)** modes.

---

## 🧠 Features
- Flame detection and automatic water spraying
- Manual control using ESP Web Server
- Dual-mode operation (Auto / Manual)
- ESP32-based control with Wi-Fi connectivity
- Real-time monitoring and control from mobile

---

## ⚙️ Hardware Components
| Component | Description |
|------------|-------------|
| ESP32 DevKit V1 | Main microcontroller |
| L298N | Motor driver module |
| SG90 Servo | Controls water nozzle direction |
| Flame Sensor (x3) | Fire detection sensors |
| Water Pump | Fire extinguishing system |
| Power Supply | 16V DC Source + LM7805 regulator |
